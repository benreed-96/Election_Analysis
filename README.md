# Election_Analysis
PROJECT OVERVIEW:
  A Colorado Board of Elections employee has given following tasks to complete election audit:
    1. Calculate total number of votes cast
    2. Complete list of candidates recieved votes
    3. Calculate total number of votes each candidate received
    4. Calculate percentage of votes each candidate won
    5. Determine winner of election based on popular vote
  ELECTION AUDIT RESULTS:
    1. How many votes were cast in this congressional election? 369,711
    2. Provide a breakdown of the number of votes and the percentage of total votes for each county in the precinct: Jefferson: 10.5% (38,855), Denver: 82.8%(306,055), Arapahoe: 6.7% (24, 801).
    3. Which county had the largest number of votes? Denver
    4. Provide a breakdown of the number of votes and the percentage of the total votes each candidate received: Charles Casper Stockham: 23.0% (85,213) Diana DeGette: 73.8% (272,892), Raymon Anthony Doane: 3.1% (11,606)
    5. Which candidate won the election, what was their vote count, and what was their percentage of the total votes? Diana DeGette 272,892 votes 73.8%
  ELECTION-AUDIT SUMMARY: 
    This script could be used for other elections. Some ways to modify this script include, but are not limited to: 
      1. Adding in other counties to be included in the final vote. 
      2. Usng population classes to determine why certain voters casted votes for a particular candidate.
